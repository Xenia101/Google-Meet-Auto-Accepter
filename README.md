## Usage

1. Chrome 설정

2. 도구 더보기

3. [확장 프로그램](chrome://extensions/)

4. 개발자 모드 On

5. 압축해제된 확장 프로그램을 로드합니다. 클릭 후 업로드

### Environment

- **Chrome Version** : 버전 83.0.4103.116(공식 빌드) (64비트) ✔
